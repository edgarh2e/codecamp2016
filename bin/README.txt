1. Ejecutar el comando:

	python convertirTGFASVG.py input.tgf

	Donde:
		* python 				= Comando para ejecutar python.
		* convertirTGFASVG.py 	= Nombre del archivo a ejecutar
		* input.tgf 			= Argumento de entrada con el nombre del archivo tgf, que debe estar en la misma ruta que el archivo .py 

2. Como salida se generan los siguientes archivos en la misma ruta:

	output.gv.svg
	output.gv
